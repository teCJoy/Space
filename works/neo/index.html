❓ 你问我答，句句实话

🚀 怎么唤醒 neo？
💻 终端里敲 python3 neo.py，神兽立马睁眼 👀
🔍 搜片姿势有哪些？
⌨️ 直接输入电影名，九站同搜；也可以把网页链接丢给它，自动帮你扒磁力 🕵️
🔄 搜完还想再来一遍？
🔁 按 r 用上次的关键词重新搜。有些站点初次容易抽风，大胆按 r 再冲一次，第二次往往能多搂一堆宝贝，亲测有效 🔄
🏃 想溜怎么办？
🚪 按 q 直接跑路，不留一片云彩 ☁️
🍪 为什么要给 neo 喂小饼干（cookie）？
🍪 B站接口认饼不认脸，没饼干它就板着脸；塞一把 Netscape 小甜饼，搜索结果唰唰现，不然白瞪眼 😤
🛠️ 小饼干怎么烤？
🍪 装个插件“Get Cookie Locally”，轻轻一点全导出，Netscape格式别搞错，放到 neo 旁就稳妥，仪式感满满真不错 ✨
📦 能捞出什么好东西？
🧲 磁力、电驴、迅雷链，种子文件加 B 站片，五种格式全囊括，一个不少随便捡 👀
⏱️ 找片速度有多狂？
💨 九个线程同时跑，比一个一个快不少，眨个眼睛结果到，慢？不存在的，快得呱呱叫 ⚡
🌍 都搜哪些藏宝阁？
🏴‍☠️ 电影港、6v、磁力狗，Sobt、磁力猫也走一走，高清族、MP4吧、SeedHub 全都有，B站视频排前头，九大门派一网收 🤺
🧩 要装一堆依赖吗？
🐍 纯 Python 标准库，不用 pip 瞎忙乎，下载即用不糊涂，轻得像片羽毛舞 🪶
🤖 碰上人机验证咋整？
🧠 MP4吧 偶尔耍脾气，加减法题考考你，neo 自动算答案，管它一题还是两题，轻松搞定不稀奇 ✏️
🛠️ 能自己加新站点吗？
🔧 代码结构很清晰，仿照格式往里递，会点正则和请求，四海站点皆归你 🧙
🔄 结果太多怎么找？
🎯 智能去重先筛一遍，B站视频置顶见，磁力电驴分类列，一目了然不迷眼 📋
💻 什么系统能驾驭？
🪟🐧🍎 Windows、Linux、macOS 全拿下，有 Python 的地方就是家，哪里需要哪里撒 🏠
